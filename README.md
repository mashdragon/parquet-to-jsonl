# parquet-to-jsonl
Convert parquet files to .jsonl in a snap

## Install requirements
```
pip install -r requirements.txt
```

## Running is easy
To run:
```
python ./parquet_to_jsonl.py train.parquet train.jsonl
```

Result:
```

```
